## Notes on Braches

Main branch is not affacted by feature branch. The changes will only be available when the merge action is taken

### commands
1. git branch -> shows the current branch by highlighting it and other branches
2. git checkout "branch name" -> switch between the branches
3. git checkout -b "branch name" -> creates new branch
